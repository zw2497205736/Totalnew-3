# LLAP 音频感知距离实现原理文档

## 项目概述

LLAP (Low Latency Acoustic Positioning) 是一个基于声学测距的 Android 应用，通过扬声器发射超声波信号，麦克风接收反射信号，利用 FMCW (调频连续波) 原理实时计算物体距离变化。

---

## 核心技术原理

### 1. FMCW 声学测距原理

项目采用**多频连续波测距**技术：
- 扬声器同时发射多个固定频率的超声波
- 麦克风接收物体反射回来的信号
- 通过分析发射信号与接收信号的**相位差变化**计算距离变化
- 相位差与距离的关系：`Δφ = 2π × 2d / λ`（往返路程）

**优势**：
- 无需脉冲发射，连续测量
- 多频率降低多普勒效应影响
- 实时性强，延迟低

### 阶段二：信号生成与播放

#### 2.1 播放缓冲区初始化 (`RangeFinder::InitBuffer`)

生成**多频率混合的参考信号**：

```cpp
void RangeFinder::InitBuffer(void) {
    for (int i = 0; i < mNumFreqs; i++) {
        for (int n = 0; n < mPlayBufferSize; n++) {
            // 生成余弦参考信号（同相分量 I）
            mCosBuffer[i][n] = cos(2 * PI * n / AUDIO_SAMPLE_RATE * mFreqs[i]);
            
            // 生成正弦参考信号（正交分量 Q）
            mSinBuffer[i][n] = -sin(2 * PI * n / AUDIO_SAMPLE_RATE * mFreqs[i]);
            
            // 累加所有频率
            mTempSample += mCosBuffer[i][n];
        }
        
        // 归一化并转换为16位整数
        mPlayBuffer[n] = (int16_t)(mTempSample / mNumFreqs * 32767);
    }
}
```

**生成的信号**：
```
s(t) = Σ cos(2π × fi × t)  (i = 1 to 10)
```
其中 fi = 18000, 18200, 18400, ..., 19800 Hz

#### 2.2 音频回调函数 (`AudioController::performRender`)

实时音频处理的核心循环：

```cpp
bool AudioController::performRender(
    void *clientdata,
    short int *audioInputOutput,
    int inNumberFrames,
    int samplerate) {
    
    CallbackData *cd = (CallbackData *) clientdata;
    
    // 步骤1: 保存麦克风输入数据
    memcpy((void *) cd->rangeFinder->GetRecDataBuffer(inNumberFrames),
           (void *) audioInputOutput,
           sizeof(int16_t) * inNumberFrames);
    
    // 步骤2: 计算距离变化量
    cd->distancechange = cd->rangeFinder->GetDistanceChange();
    
    // 步骤3: 累积距离（带速度校准）
    cd->distance = cd->distance + cd->distancechange * SPEED_ADJ;
    
    // 步骤4: 距离限幅 [0, 500]
    if (cd->distance < 0) cd->distance = 0;
    if (cd->distance > 500) cd->distance = 500;
    
    // 步骤5: 输出播放信号到扬声器
    memcpy((void *) audioInputOutput,
           (void *) cd->rangeFinder->GetPlayBuffer(inNumberFrames),
           sizeof(int16_t) * inNumberFrames);
    
    // 步骤6: UI更新（通过JNI回调到Java层）
    if (fabs(cd->distancechange) > 0.06 && 
        (startTime - cd->mUIUpdateTime) / 1.0e6 > 10) {
        JNIEnv *env;
        (*gs_jvm).AttachCurrentThread(&env, NULL);
        (*env).CallVoidMethod(mObject, gOnNativeID, int(cd->distance));
        cd->mUIUpdateTime = startTime;
    }
    
    return true;
}
```

**处理流程**：
```
麦克风输入 → 保存数据 → 计算距离变化 → 累积距离 → 输出播放信号 → 更新UI
   ↑                                                      ↓
   └──────────────── 循环 (每 10.67ms) ──────────────────┘
```

### 阶段三：信号处理与解调

#### 3.1 基带信号提取 (`RangeFinder::GetBaseBand`)

使用**正交解调 + CIC 滤波器**提取各频率的复数基带信号：

##### 3.1.1 正交解调

对每个频率 fi 进行 I/Q 解调：

```cpp
for (int i = 0; i < mNumFreqs; i++) {
    // I路（同相分量）：与 cos(2πfit) 相乘
    DSP_vmul(
        mFRecDataBuffer,              // 接收信号
        mCosBuffer[i] + mCurProcPos,  // 余弦参考
        mTempBuffer,                   // 输出缓冲
        mCurRecPos                     // 数据长度
    );
    
    // Q路（正交分量）：与 -sin(2πfit) 相乘
    DSP_vmul(
        mFRecDataBuffer,              // 接收信号
        mSinBuffer[i] + mCurProcPos,  // 正弦参考
        mTempBuffer,                   // 输出缓冲
        mCurRecPos                     // 数据长度
    );
}
```

**原理**：
```
I(t) = r(t) × cos(2πfit) = A·cos(φ)  (滤除高频)
Q(t) = r(t) × (-sin(2πfit)) = A·sin(φ)  (滤除高频)
```

##### 3.1.2 CIC 滤波器（四级级联）

**CIC (Cascaded Integrator-Comb)** 滤波器实现抽取和低通滤波：

```cpp
// 抽取倍数
int CIC_DEC = 16;  // 48000 Hz → 3000 Hz

// 第一级：累加求和（积分器）
for (int k = 0; k < decsize; k++) {
    DSP_sve(
        mTempBuffer + k,                // 输入：每16个点
        mCICBuffer[i][0][cid] + index,  // 输出：累加和
        CIC_DEC                         // 累加个数
    );
}

// 第二级到第四级：滑窗求和（梳状滤波器）
DSP_vswsum(mCICBuffer[i][0][cid], mCICBuffer[i][1][cid]+CIC_DELAY, 
           decsize, CIC_DELAY);
DSP_vswsum(mCICBuffer[i][1][cid], mCICBuffer[i][2][cid]+CIC_DELAY, 
           decsize, CIC_DELAY);
DSP_vswsum(mCICBuffer[i][2][cid], mCICBuffer[i][3][cid]+CIC_DELAY, 
           decsize, CIC_DELAY);
DSP_vswsum(mCICBuffer[i][3][cid], mBaseBandReal[i], 
           decsize, CIC_DELAY);
```

**滤波效果**：
- 采样率：48000 Hz → 3000 Hz
- 通带：0 ~ 150 Hz（保留多普勒频移范围）
- 阻带衰减：>80 dB

#### 3.2 直流分量去除 (`RangeFinder::RemoveDC`)

由于硬件非线性和环境噪声，基带信号含有直流偏移，需要自适应去除：

```cpp
void RangeFinder::RemoveDC(void) {
    for (int f = 0; f < mNumFreqs; f++) {
        float vsum = 0, dsum = 0;
        
        // 计算功率
        for (int i = 0; i < mDecsize; i++) {
            // 方差
            DSP_vsadd(mBaseBandReal[f], &temp_val, tempdata, mDecsize);
            DSP_sve(tempdata, &temp_val, mDecsize);
            vsum = vsum + fabs(temp_val) / mDecsize;
            
            // 均值
            DSP_sve(mBaseBandReal[f], &temp_val, mDecsize);
            dsum = dsum + fabs(temp_val) / mDecsize;
        }
        
        mFreqPower[f] = vsum + dsum * dsum;
        
        // 只处理有效信号（功率阈值）
        if (mFreqPower[f] > POWER_THR) {
            // Levd算法：DC = (max + min) / 2
            mDCValue[i][f] = (1 - DC_TREND) * mDCValue[i][f] + 
                             (mMinValue[i][f] + mMaxValue[i][f]) / 2 * DC_TREND;
        }
        
        // 减去DC分量
        for (int i = 0; i < mDecsize; i++) {
            mBaseBandReal[f][i] -= mDCValue[0][f];
            mBaseBandImag[f][i] -= mDCValue[1][f];
        }
    }
}
```

**算法要点**：
- **功率检测**：`POWER_THR = 0.001`，过滤弱信号
- **Levd算法**：假设信号对称分布，DC ≈ (峰值 + 谷值) / 2
- **自适应跟踪**：`DC_TREND = 0.01`，平滑更新

### 阶段四：距离计算

#### 4.1 相位解算 (`RangeFinder::CalculateDistance`)

##### 4.1.1 计算瞬时相位

```cpp
for (int f = 0; f < mNumFreqs; f++) {
    for (int i = 0; i < mDecsize; i++) {
        // 复数转相位：φ = atan2(Q, I)
        phasedata[f][i] = atan2(
            mBaseBandImag[f][i],  // Q路（虚部）
            mBaseBandReal[f][i]   // I路（实部）
        );
    }
}
```

**相位范围**：[-π, π]

##### 4.1.2 相位解缠绕（Unwrap）

防止相位跳变导致的距离突变：

```cpp
for (int i = 1; i < mDecsize; i++) {
    // 向上跳变：-π → π（差值 > π）
    while (phasedata[f][i] - phasedata[f][i-1] > PI)
        phasedata[f][i] -= 2 * PI;
    
    // 向下跳变：π → -π（差值 < -π）
    while (phasedata[f][i] - phasedata[f][i-1] < -PI)
        phasedata[f][i] += 2 * PI;
}
```

**效果**：
```
解缠前: ..., 3.0, -3.1, -2.9, ...  (跳变)
解缠后: ..., 3.0,  3.2,  3.4, ...  (连续)
```

##### 4.1.3 相位转距离

```cpp
// 波长计算：λ = c / f
mWaveLength[f] = mSoundSpeed / mFreqs[f];

// 相位梯度转距离梯度
temp_val = 2 * PI / mWaveLength[f];  // 2π/λ
for (int i = 0; i < mDecsize; i++) {
    tempdata[i] = phasedata[f][i] - phasedata[f][0];  // 相对相位
    phasedata[f][i] = tempdata[i] / temp_val;         // 转换为距离
}
```

**公式**：
```
d = Δφ × λ / (2π × 2)
  = Δφ / (2π/λ × 2)
```
其中因子 2 来自往返路程。

#### 4.2 最小二乘线性回归

由于单频率存在周期性模糊，使用**多频率最小二乘拟合**提高精度：

```cpp
float CalculateDistance(void) {
    float delta = 0;
    int numfreqused = 0;
    float sumxy = 0, sumy = 0;
    
    // 时间轴
    float deltax = (mDecsize - 1) * mDecsize / 2.0;
    
    for (int f = 0; f < mNumFreqs; f++) {
        if (mFreqPower[f] > POWER_THR) {
            numfreqused++;
            
            // Σ(xi × yi)
            for (int i = 0; i < mDecsize; i++) {
                sumxy += i * phasedata[f][i];
            }
            
            // Σyi
            DSP_sve(phasedata[f], &temp_val, mDecsize);
            sumy += temp_val;
        }
    }
    
    // 最小二乘斜率：k = (Σxy - Σy·x̄) / Σ(x - x̄)²
    delta = (sumxy - sumy * (mDecsize - 1) / 2.0) / deltax * 
            mNumFreqs / numfreqused;
    
    // 距离 = 斜率 × 时间中点
    distance = -delta * mDecsize / 2;
    
    return distance;
}
```

**线性回归模型**：
```
距离序列: d[i] = d0 + v·ti  (v为速度)
最小化: Σ(d[i] - d̂[i])²
求解: v = (Σxiyi - Σxi·Σyi/n) / (Σxi² - (Σxi)²/n)
距离变化: Δd = v × Δt
```

## 局限性与改进方向

### 当前局限
1. **多径干扰**：室内复杂环境下易受多径反射影响
2. **温度敏感**：声速随温度变化 (~0.6 m/s/°C)
3. **角度依赖**：非正向反射时精度下降
4. **距离范围**：受超声波衰减限制，有效距离约5米

### 改进方向
1. **温度补偿**：集成温度传感器实时校准声速
2. **波束成形**：使用麦克风阵列提高方向性
3. **卡尔曼滤波**：对距离序列进行平滑滤波
4. **机器学习**：训练模型补偿非线性误差
5. **自适应频率**：根据环境噪声动态调整频率

## 附录：关键公式汇总

### 1. 相位-距离关系
```
Δφ = 2π × 2d / λ
d = Δφ × λ / (4π)
```

### 2. 声速-温度关系
```
c = 331.3 + 0.606 × T  (m/s)
```

### 3. I/Q解调
```
I(t) = LPF[r(t) × cos(2πft)]
Q(t) = LPF[r(t) × (-sin(2πft))]
φ = atan2(Q, I)
```

### 4. CIC传递函数
```
H(z) = [(1 - z^(-RM)) / (1 - z^(-1))]^N
```
其中 R=抽取率, M=差分延迟, N=级数

### 5. 最小二乘估计
```
v̂ = (Σxiyi - n·x̄·ȳ) / (Σxi² - n·x̄²)
Δd = v̂ × Δt
```
