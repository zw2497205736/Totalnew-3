💡 理论分析
============================================================

【相位解包装原理】

1. 相位-距离关系:
   Δφ = 2π × Δd / λ
   
2. 相位测量范围: -π ~ +π (atan2函数)

3. 最大可追踪距离（单帧）:
   
   ✅ 理论极限 (λ/2):
      当 Δφ = π 时，Δd_max = λ/2
      推导: π = 2π × Δd_max / λ  →  Δd_max = λ/2
      
      对于20kHz超声波 (λ=17.15mm):
      - 单帧最大追踪: 8.57mm
      - 最大检测速度: 8.57mm × 47.6fps = 408mm/s = 40.8cm/s
   
   ⚠️ 保守阈值 (λ/4):
      考虑噪声、多径反射等实际因素，使用更保守的λ/4
      - 单帧最大追踪: 4.29mm
      - 最大检测速度: 4.29mm × 47.6fps = 204mm/s = 20.4cm/s

【失效判断标准（重要修正）】

❌ 错误判断1: 距离值接近0 = 失效
   → 距离为0可能是回到初始位置（正常）

❌ 错误判断2: 仅根据速度超限判断
   → 速度未超限也可能失效（噪声/信号质量问题）

✅ 正确判断: 距离从大值突然跳变到0 + 排除能量重置
   → 前一帧 |d| > 50mm，当前帧 |d| < 10mm，变化 > 40mm
   → 且能量 < 1.2 (排除能量重置机制)
   → 例如: -100mm → -5mm，能量1.01 → 1.13 ❌ LLAP失效
   → 反例: -60mm → -4mm，能量1.02 → 1.30 ✅ 能量重置(正常)

【失效机制】

当物体移动速度超过阈值时:
1. 相邻帧相位变化 > π (或π/2保守)
2. atan2无法区分2π整数倍的相位跳变
3. 相位解包装算法失效
4. 累积距离出现非物理的大幅跳变（瞬时速度异常高）

【优化效果对比】

优化前 (25fps):
  - 理论极限(λ/2): 8.57 × 25 = 214mm/s = 21.4cm/s
  - 保守阈值(λ/4): 4.29 × 25 = 107mm/s = 10.7cm/s

优化后 (47.6fps):
  - 理论极限(λ/2): 8.57 × 47.6 = 408mm/s = 40.8cm/s ⬆️1.9倍
  - 保守阈值(λ/4): 4.29 × 47.6 = 204mm/s = 20.4cm/s ⬆️1.9倍

============================================================
🎯 公式确认
============================================================

✅ 最大检测速度公式:

理论极限: v_max = (λ/2) × fps
保守实用: v_max = (λ/4) × fps

其中:
- λ = 声速 / 频率 = 343m/s / 20000Hz = 17.15mm
- fps = LLAP处理帧率

推导依据:
- 代码中使用 distance = phase / (2π/λ)
- 相位范围 -π~π，最大变化π对应距离λ/2
- 实际应用中考虑噪声，使用λ/4更稳健

失效判断标准:
- 计算相邻帧距离变化: Δd = d[i] - d[i-1]
- 计算时间间隔: Δt = t[i] - t[i-1]
- 瞬时速度: v = |Δd| / Δt
- 失效条件: v > (λ/2) × fps
